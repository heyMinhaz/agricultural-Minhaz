# Agricultural Information Hub

A basic responsive static website with farming guides, crop details, pest control strategies, and a contact form.

## Run Locally

Open `index.html` in a browser.

## Deploy

This project is ready for Netlify static hosting. No build command is required.

### Netlify Drop

1. Go to `https://app.netlify.com/drop`
2. Upload this project folder or `agricultural-information-hub.zip`
3. Netlify will provide a live URL

### Netlify Settings

Build command: leave empty
Publish directory: `.`
